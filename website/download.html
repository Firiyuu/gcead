Installation programs are available for Windows and Mac. Please see the appropriate section below.

Microsoft Windows

NOTE: The software has only been tested on Windows XP (32-bit) and Windows 7 (64-bit).
  1. Download and run the installer: GcEad-1.2.5-windows-installer-20140503.exe
  2. If you want to use older Syntech software (such as AutoSpike) on the same computer as GcEad/2014, you should probably install them prior to installing GcEad/2014. This is because if you select to install the USB drivers that come with GcEad/2014 before the older software has already been installed, then the older software will not be able to access the IDAC USB hardware.
  3. If you want to use the IDAC-2 or IDAC-4 acquisition hardware with GcEad/2014, and you don't require older Syntech software, then please choose to install the USB driver during installation. Full installation instructions can be found at this link.

Mac OS X

NOTE: The software has only been tested on version 10.7 of OS X.
  1. Download GcEad-1.2.5-mac-installer-20140503.dmg
  2. You can now run GcEad by double-clicking on the .dmg file
