For Microsoft Windows

WARNING: At this time (2008-09-26) there are unfortunately difficulties with the installation procedure, which will require that you obtaining drivers from Syntech in order for the program to run.

First download this file: GcEad2008-20080919.zip

Once downloaded, please unzip the file, go to the "20080919-Release" sub-directory, and run setup.exe to install GC-EAD 2008.

Source Code

The source code can be downloaded from the sourceforge SVN server using this url:
https://gcead.svn.sourceforge.net/svnroot/gcead/trunk

If you are using the command-line svn utility, then the following command line should word:
svn co https://gcead.svn.sourceforge.net/svnroot/gcead/trunk gcead
